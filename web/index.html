EfficientNet-B0

A 5.3M-parameter backbone pretrained on ImageNet. Two-phase fine-tuning: frozen backbone for 3 epochs, then full fine-tuning at a discriminative learning rate.

Grad-CAM Explanation

Gradients of the predicted class w.r.t. the last convolutional block produce a saliency map. We overlay it on your image so you see exactly what the model focused on.

⚡

Two-Phase Training

Backbone frozen for 3 epochs while the head learns fast. Then unfrozen at lr÷5 with cosine annealing. AUC jumped from 0.876 → 0.915 in a single epoch post-unfreeze.

🎯

Grad-CAM

Gradients of the class score w.r.t. blocks[-1] output are globally average-pooled → importance weights per channel → weighted feature map sum → bilinear upscale to 224px.

🌡️

Temperature Scaling

A single scalar T optimised on validation NLL. Calibrated logits = logits ÷ T. No accuracy change — only confidence alignment. Evaluated with Expected Calibration Error (15 bins).

🔮

MC Dropout

At inference, Dropout stays active. 30 stochastic passes sample different network masks. Predictive entropy H = −Σ p̄ log p̄ quantifies epistemic uncertainty.

About the Author

Why calibration
matters more
than accuracy

Built end-to-end by Toumi Rayane

Most student ML projects report accuracy on balanced test sets and stop there. But in medical AI, confidence calibration is what separates a research demo from something deployable.

A model that says "97% melanoma" when it should say "62%" will cause harm — either unnecessary panic or false reassurance. Temperature scaling is a one-line fix with provable guarantees on the validation set.

Similarly, uncertainty quantification via MC Dropout means the model can say "I don't know" — flagging ambiguous cases for human review rather than forcing a confident wrong answer.
